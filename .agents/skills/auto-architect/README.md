# AutoArchitect Skills

Este conjunto de skills foi criado para analisar e documentar sistemas automaticamente a partir do código-fonte.

## 🧠 Como funciona

O processo é dividido em duas etapas:

### 1. AutoArchitect.Scan

Primeiro você deve executar essa skill.

**Função:**

* Analisa todo o sistema
* Entende a arquitetura
* Identifica regras de negócio
* Mapeia banco de dados e cálculos
* Constrói um modelo mental completo

⚠️ **Importante:**
Essa etapa NÃO gera documentação final. Ela apenas prepara o entendimento do sistema.

---

### 2. AutoArchitect.Doc

Depois de executar o Scan, utilize essa skill.

**Função:**

* Gera a documentação completa em `.md`
* Organiza frontend, backend e banco
* Explica regras de negócio
* Cria guia de recriação do sistema

---

## 🔄 Ordem correta de uso

Sempre execute nesta sequência:

1. `AutoArchitect.Scan`
2. `AutoArchitect.Doc`

---

## 🚨 Observações

* Não pule a etapa de Scan
* A qualidade da documentação depende diretamente da análise inicial
* O processo foi dividido para evitar perda de contexto e respostas superficiais

---

## 🎯 Resultado final

Ao seguir esse fluxo, você terá:

* Um documento completo do sistema
* Explicação detalhada de funcionamento
* Guia para recriar o projeto do zero
